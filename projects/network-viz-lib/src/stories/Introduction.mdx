## ğŸš€ Quick Start

```typescript
import { NetworkVisualizationV2Component } from '@your-org/network-viz-lib';

@Component({
  template: `
    <nvl-network-visualization-v2
      [data]="networkData"
      [config]="config"
      [width]="800"

    nodes: [
      { id: 1, label: 'Node 1', group: 'A' },
      { id: 2, label: 'Node 2', group: 'B' }

    links: [
      { source: 1, target: 2 }

      defaultSize: 15,

    }
```
